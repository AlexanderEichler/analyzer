Name : Base
Input : list of .c files : "A C-Program or a fragment thereof."
Info : "Constant propagation info and a mulit-threadedness flag"
Preprocessing : no 
Interval Domain : yes
                    [ ] Trier ; "Concrete integers (3) or exclusions sets (not {0,1})"     
                    [ ] Interval ; "Where we possibly know the maximum and minimum of a variable." 
Context Sensitive : default yes
Path Sensitive : default no
Options : 
  --allfuns :  Analyzes all the functions (not just beginning from main).
  --noverify :  Skip the verification phase.
  --nonstatic : Analyzes all non-static functions.
  --mainfun : Sets the name of the main function.
  --exitfun : Sets the name of the main function.
  --field-insensitive : Turns off field-sensitivity.
  --kernel : For analyzing Linux Device Drivers.
  --type-inv <bool> : Should we use type invariants?
Result : format string for errors ; 
        "Errors when not supported constructs are used (inline-assembly, pointer arithmetic, etc)"
        "Unsoundness waringins --- when when we are too imprecise, but instead of saying that we do not know anything, we assume that nothing bad happened."
         format string for warnings ; 
         format string for misc messages ; 
Dependencies : -
Solver : if interval are used then solverConSideWNRR else All
Goblint : --analyis base (not modifiable for the users)
